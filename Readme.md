In this game, players can first read the game description on the first screen and enter their names. Then they can control a robot to move and avoid obstacles and mines to save people. They can also switch to the AI mode to do these tasks. Every time they successfully save a person, they will earn 10 score. Every time they save 5 people, their level will increase. The higher the level, the faster the game speed. Finally, I made some innovations. For example, the life of the robot is equal to the number of their body segments. after robot died, it has invincibility time. The robot can destroy some landmines by consuming levels.
